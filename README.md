# HouseHub (House Harmony)

## Overview
HouseHub is a web application designed to help people who live together manage shared responsibilities in an organized and transparent way. In shared houses, chores like cleaning or buying essential supplies are often managed manually, leading to confusion. HouseHub provides a simple digital system to track these tasks and maintain fairness.

## Features
- **House Management**: Create a house and invite members using a unique 6-digit house code.
- **Join House**: Seamlessly join an existing house by entering the house code.
- **Cleaning Rotations**: Automated weekly cleaning rotation with scheduled days (Saturday).
- **Completion Tracking**: Record when members complete their assigned cleaning duties.
- **Household Supplies**: Track shared purchases such as Gas, Water, Soap, and Sponges.
- **Purchase History**: View who last purchased specific items to ensure transparency.
- **Dashboard**: A central view showing recent activity and upcoming responsibilities.
- **Alert Reminders**: Notifications for upcoming tasks.

## Tech Stack
- **React**: UI Library
- **TypeScript**: Type Safety
- **Vite**: Build Tool
- **Tailwind CSS**: Styling
- **shadcn-ui**: UI Components

## Installation
To run the project locally, follow these steps:

1. **Clone the repository**:
   ```sh
   git clone <repository-url>
   ```

2. **Navigate to the project folder**:
   ```sh
   cd house-harmony-44
   ```

3. **Install dependencies**:
   ```sh
   npm install
   ```

4. **Run the development server**:
   ```sh
   npm run dev
   ```

## Usage
Once the development server is running, open your browser to the provided local URL. You can start by creating a new house or joining one with a 6-digit code. From the dashboard, you can manage cleaning schedules and track household supplies.

## Project Goal
The goal of HouseHub is to provide a lightweight, easy-to-use tool that helps housemates stay organized and avoid misunderstandings regarding shared responsibilities.
